HTML

Datos importantes en TecWeb
Numero Elemento Proposito Ejemplo
1 div Contenedor genérico del contenido de flujo. No afecta al contenido ni al diseño hasta que se le aplique algún estilo mediante CSS. <div> <p>Hola</p></div>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
2 header El elemento representa contenido introductorio, generalmente un grupo de ayudas introductorias o de navegación. Puede contener elementos de encabezado. <header> <a class="logo" href="#">Cute Puppies Express!</a> </header>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
3 nav Define una sección de navegación con enlaces a otras partes del sitio. <nav> <a href="index.html">Inicio</a> <a href="contacto.html">Contacto</a> </nav>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
4 p Representa un párrafo de texto. <p>Este es un párrafo de ejemplo.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
5 ul Elemento que representa una lista desordenada de elementos, normalmente representada con viñetas. <ul> <li>Elemento 1</li> <li>Elemento 2</li> </ul>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
6 li Elemento que representa un ítem dentro de una lista. <li><a href="index.html">HTML</a></li>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
7 a Define un enlace a otra página o recurso. <a href="index.html">Ir a Inicio</a>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
8 head Contiene metadatos y enlaces a recursos como CSS y scripts. <head><title>Mi Página</title></head>
9 body Define el contenido principal de la página web. <body><p>Hola, mundo</p></body>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
10 footer Define el pie de página de un documento o sección. <footer>Derechos reservados 2025</footer>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
11 img Incorpora imágenes en la página. <img src="imagen.jpg" alt="Descripción">
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
12 table Define una tabla de datos. <table><tr><td>Dato 1</td></tr></table>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
13 section Define una sección en un documento. <section><h2>Sección 1</h2><p>Contenido</p></section>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
14 article Representa un contenido independiente y autocontenido, como una noticia o publicación. <article><h2>Noticia</h2><p>Contenido de la noticia</p></article>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
15 aside Define contenido relacionado pero independiente del contenido principal, como una barra lateral. <aside><p>Publicidad aquí</p></aside>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
16 form Define un formulario para enviar datos. <form><input type="text" name="nombre"><button>Enviar</button></form>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
17 input Define un campo de entrada de datos en un formulario. <input type="email" placeholder="Ingresa tu correo">
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
18 button Representa un botón interactivo. <button>Haz clic</button>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
19 label Asocia un texto descriptivo con un elemento de formulario. <label for="nombre">Nombre:</label><input id="nombre" type="text">
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
20 select Define un menú desplegable. <select><option>Opción 1</option><option>Opción 2</option></select>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
21 textarea Crea un área de texto multilínea para ingresar datos. <textarea rows="4" cols="50">Escribe aquí</textarea>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
22 meta Define metadatos para la página, como la codificación de caracteres. <meta charset="UTF-8">
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
23 h1 Define un encabezado de nivel 1, el más importante en la jerarquía. <h1>Título Principal</h1>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
24 h2 Encabezado de segundo nivel. <h2>Subtítulo</h2>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
25 h3 Encabezado de tercer nivel. <h3>Sección menor</h3>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
26 h4 Encabezado de cuarto nivel. <h4>Encabezado pequeño</h4>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
27 h5 Encabezado de quinto nivel. <h5>Encabezado menor</h5>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
28 h6 Encabezado de sexto nivel, el de menor importancia. <h6>Encabezado más pequeño</h6>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
29 br Inserta un salto de línea. <p>Línea 1<br>Línea 2</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
30 hr Inserta una línea horizontal para separar contenido. <hr>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
31 wbr Indica un punto donde puede ocurrir un salto de línea opcional. <p>EjemploDe<wbr>PalabraLarga</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
32 sup Coloca el texto como superíndice. <p>X<sup>2</sup></p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
33 sub Coloca el texto como subíndice. <p>H<sub>2</sub>O</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
34 strong Indica texto con énfasis fuerte (negrita). <p>Este es un <strong>mensaje importante</strong>.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
35 em Indica texto con énfasis (generalmente en cursiva). <p>Este es un <em>mensaje enfático</em>.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
36 mark Resalta texto, generalmente con un fondo amarillo. <p>Este es un <mark>texto resaltado</mark>.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
37 q Representa una cita breve. <p>Ella dijo: <q>Hola</q>.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
38 blockquote Define una cita larga o bloque de cita. <blockquote>Este es un texto citado.</blockquote>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
39 cite Indica el título de una obra citada. <p>Leí <cite>El Quijote</cite>.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
40 dfn Marca una palabra como término definido. <p>El término <dfn>HTML</dfn> significa Lenguaje de Marcado de Hipertexto.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
41 abbr Define una abreviatura o acrónimo. <p><abbr title="Organización de las Naciones Unidas">ONU</abbr></p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
42 small Representa texto con menor importancia. <p>Aviso legal: <small>Texto en pequeño</small>.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
43 ins Indica texto insertado o agregado. <p>Este es un <ins>nuevo texto</ins>.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
44 del Indica texto eliminado. <p>Este es un <del>texto eliminado</del>.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
45 s Indica texto tachado, generalmente para mostrar contenido obsoleto. <p>El precio anterior era <s>$100</s>, ahora es $80.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
46 code Representa código fuente en línea. <p>Usa <code>console.log('Hola')</code></p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
47 samp Representa salida de un programa o computadora. <p>El resultado es: <samp>404 Not Found</samp>.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
48 kbd Indica entrada de teclado. <p>Presiona <kbd>Ctrl + C</kbd> para copiar.</p>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
49 pre Representa texto preformateado con espacios y saltos de línea. <pre> Linea 1 Linea 2</pre>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
50 address Define información de contacto. <address>Contacto: contacto@ejemplo.com</address>
_______________________________________________________________ _______________________________________________________________ _______________________________________________________________ _______________________________________________________________
51 span Contenedor en línea para aplicar estilos o scripts. <p>Este es un <span style="color:red">texto rojo</span>.</p>
