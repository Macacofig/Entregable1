HTML

Numero Elemento Proposito Ejemplo
1 div Contenedor genérico de bloque utilizado para agrupar contenido y aplicar estilos o scripts. <div class="contenedor">Contenido</div>
2 span Contenedor en línea usado para agrupar texto y aplicar estilos sin afectar el flujo del documento. <span style="color:red;">Texto en rojo</span>
3 header Define el encabezado de una sección o página, que puede contener títulos, logotipos y navegación. <header><h1>Mi Sitio</h1></header>
4 article Representa contenido independiente y reutilizable, como un artículo de blog o una noticia. <article><h2>Noticia</h2><p>Texto...</p></article>
5 section Define una sección de contenido temáticamente relacionada dentro de una página. <section><h2>Tema</h2><p>Descripción</p></section>
6 nav Contiene enlaces de navegación principales del sitio web. <nav><ul><li><a href="#">Inicio</a></li></ul></nav>
7 aside Define contenido relacionado o complementario, como barras laterales o anuncios. <aside><p>Publicidad</p></aside>
8 footer Representa el pie de página de una sección o documento, que suele contener información de copyright o enlaces adicionales. <footer>© 2025 Mi Sitio</footer>
9 main Indica el contenido principal del documento, excluyendo encabezados, menús, pie de página, etc. <main><h1>Contenido principal</h1></main>
10 h1 Define el título más importante del documento (solo debe haber un <h1> por página). <h1>Bienvenidos</h1>
11 h2 Define un subtítulo de segundo nivel, menor en importancia que <h1>. <h2>Sección principal</h2>
12 h3 Encabezado de tercer nivel, normalmente usado para subsecciones dentro de <h2>. <h3>Subtema</h3>
13 h4 Encabezado de cuarto nivel, menos relevante que <h3>`. <h4>Detalles</h4>
14 h5 Encabezado de quinto nivel, poco utilizado. <h5>Información adicional</h5>
15 h6 Encabezado de sexto nivel, el de menor jerarquía. <h6>Nota secundaria</h6>
16 p Define un párrafo de texto. Se usa para estructurar contenido en bloques. <p>Este es un párrafo.</p>
17 br Inserta un salto de línea dentro del texto. No tiene etiqueta de cierre. Texto <br> Nueva línea
18 hr Dibuja una línea horizontal para separar secciones. <hr>
19 wbr Sugiere un punto de salto de línea si es necesario en palabras largas. Supercalifragilist<wbr>icoespialidoso
20 sup Muestra texto en superíndice, útil para exponentes matemáticos. X<sup>2</sup>
21 sub Muestra texto en subíndice, útil en fórmulas químicas. H<sub>2</sub>O
22 strong Indica texto con énfasis fuerte, normalmente mostrado en negrita. <strong>Importante</strong>
23 em Indica texto con énfasis, normalmente mostrado en cursiva. <em>Destacado</em>
24 mark Resalta texto con un fondo de color. <mark>Texto resaltado</mark>
25 q Cita en línea breve con comillas automáticas. <q>Cita corta</q>
26 blockquote Define una cita en bloque con sangría. <blockquote>Texto citado en bloque.</blockquote>
27 cite Indica la fuente de una cita. <cite>Nombre del autor</cite>
28 dfn Define un término o concepto. <dfn>HTML</dfn> significa HyperText Markup Language.
29 abbr Indica una abreviatura o acrónimo con una descripción emergente. <abbr title="HyperText Markup Language">HTML</abbr>
30 small Reduce el tamaño del texto, útil para notas legales. <small>Aviso legal</small>
31 ins Indica texto insertado o agregado recientemente. <ins>Nuevo contenido</ins>
32 del Indica texto eliminado o incorrecto. <del>Texto tachado</del>
33 s Texto tachado sin significado semántico específico. <s>Texto incorrecto</s>
34 code Muestra fragmentos de código fuente. <code>console.log('Hola')</code>
35 samp Muestra salida de un programa o consola. <samp>Error 404</samp>
36 kbd Representa una entrada de teclado. <kbd>Ctrl + C</kbd>
37 pre Mantiene los espacios y saltos de línea en el contenido. <pre>Texto alineado</pre>
38 address Define información de contacto. <address>Email: info@ejemplo.com</address>
39 body Contiene el contenido visible de la página web. <body>Contenido aquí</body>
40 title Define el título de la página web en la pestaña del navegador. <title>Mi Página</title>
41 meta Define metadatos del documento (codificación, descripción, etc.). <meta charset="UTF-8">
42 head Contiene información sobre el documento, como estilos y scripts. <head><title>Título</title></head>
43 html Elemento raíz del documento HTML. <html lang="es">...</html>
44 !DOCTYPE Define el tipo de documento (HTML5 en este caso). <!DOCTYPE html>
45 img Inserta imágenes en la página. <img src="imagen.jpg" alt="Descripción">
46 ul Crea una lista desordenada con viñetas. <ul><li>Elemento 1</li></ul>
47 li Define un elemento dentro de una lista. <li>Elemento de lista</li>
48 table Define una tabla de datos. <table>...</table>
49 caption Agrega un título descriptivo a una tabla. <caption>Datos de ventas</caption>
50 thead Define la cabecera de una tabla. <thead><tr><th>Columna</th></tr></thead>
51 tbody Define el cuerpo de la tabla. <tbody><tr><td>Dato</td></tr></tbody>
52 tfoot Define el pie de la tabla. <tfoot><tr><td>Total</td></tr></tfoot>
53 tr Define una fila dentro de la tabla. <tr><td>Celda</td></tr>
54 td Define una celda dentro de una tabla. <td>Dato</td>
55 a Crea un enlace. <a href="https://google.com">Google</a>
56 link Vincula recursos externos, como hojas de estilo. <link rel="stylesheet" href="style.css">
